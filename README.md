# Factorials
Calculate factorials of large numbers.  
Runs stable up to 100,000,000 for now.

### Build
Requires [GNU Multiple Precision Library](https://gmplib.org/ "GMP Homepage").  
Make sure to configure GMP
- ```"./configure --enable-cxx"```
.

#### Build command:

- ```make```
